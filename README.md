## ‚öôÔ∏è Installation

Make sure you have [Node.js](https://nodejs.org/en/download/) installed.
Run this followed commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:3000
npm run dev

# Build for production
npm run build
```
